       - `staticDir: 'public/media'` - Local filesystem storage
       - `imageSizes` array:
         - thumbnail: 200x200, position: 'centre'
         - small: 400x300, position: 'centre'
         - card: 768x512, position: 'centre'
         - large: 1280x720, position: 'centre'
         - hero: 1920x1080, position: 'centre'
       - `adminThumbnail: 'card'`

       - NOTE: Disable SVG upload for security

    3. **Fields:**
       - alt (text, required) - admin description: 'Alternative text for accessibility (required)'
       - caption (text) - admin description: 'Image caption or credit'

       - Full access for authenticated users (admin only)
       - create, read, update, delete all require authentication
