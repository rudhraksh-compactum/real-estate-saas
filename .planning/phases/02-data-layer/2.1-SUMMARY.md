### Access Control

```typescript

  update: ({ req: { user } }) => Boolean(user),  // Admin only

```

2. Valid `PAYLOAD_SECRET` environment variable
3. Database connection configured in `payload.config.ts`

Run migration when infrastructure is ready:
```bash
docker-compose up -d
```
