## Configuration Details

### Media Collection Features
- **Slug:** `media`
- **Storage:** Local filesystem (`public/media/`)
- **Image Sizes Generated:**
  - `thumbnail`: 200x200
  - `small`: 400x300
  - `card`: 768x512
  - `large`: 1280x720
  - `hero`: 1920x1080
- **Admin Thumbnail:** `card`
- **MIME Types:** `image/jpeg`, `image/png`, `image/webp`, `image/gif` (SVG excluded for security)

### Fields
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `alt` | text | Yes | Alternative text for accessibility |
| `caption` | text | No | Image caption or credit |

### Access Control
- Admin-only access enforced (all operations require authenticated user)
- `create`, `read`, `update`, `delete` all check for `Boolean(user)`

## Threat Mitigations

| Threat | Mitigation |
|--------|------------|
| SVG-based XSS | SVG mime type excluded |
| Malicious file upload | MIME type validation restricts to images only |
| Large file storage | File size limits configurable via Payload |
