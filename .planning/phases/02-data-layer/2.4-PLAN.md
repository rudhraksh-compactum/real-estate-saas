       - email (email, required)

       - description (textarea) - admin description: 'About the business'
       - logo (upload, relationTo: 'media')
       - socialLinks (group):
         - instagram (text)
         - facebook (text)
         - twitter (text)
         - linkedin (text)
         - youtube (text)
       - address (group):
         - street (text)
         - city (text)
         - state (text)
         - zipCode (text)
         - country (text)

    Export the collection as `Accounts`.
  </action>
