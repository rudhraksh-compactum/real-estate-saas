# Phase 3 - Plan 3.3: Activity Inquiry

**Status:** Complete
**Executed:** 2026-06-01
**Wave:** 2

## Summary

Created activity inquiry form component and added activityReference to Leads collection for capturing activity booking leads.

## What Was Built

### Leads Collection Update (`payload/src/collections/Leads.ts`)
- Added `activityReference` relationship field to Activities collection
- Both propertyReference and activityReference are optional (lead can come from either)

### Zod Validation Schema Extension (`frontend/src/lib/schemas.ts`)
- Added `activityInquirySchema`: Same structure as inquirySchema but with activityReference
- `ActivityInquiryFormData` type exported

### Server Action Extension (`frontend/src/lib/actions.ts`)
- Added `submitActivityInquiry`: Server Action that:
  - Validates input with activityInquirySchema
  - Creates lead with activityReference and source: 'activity_inquiry'

### React Component (`frontend/src/components/ActivityInquiryForm.tsx`)
- ActivityInquiryForm component with:
  - Props: activityId (required), activityTitle (optional)
  - Same form structure as InquiryForm
  - Tailwind CSS styling consistent with InquiryForm
  - Calls submitActivityInquiry instead of submitInquiry
