# Phase 3 - Plan 3.1: Activities Collection

**Status:** Complete
**Executed:** 2026-06-01
**Wave:** 1

## Summary

Created the Activities collection for Airbnb experiences that can be linked to properties.

## What Was Built

### Activities Collection (`payload/src/collections/Activities.ts`)
- Full Payload CMS collection with slug: 'activities'
- Fields organized in 5 tabs: Basic Info, Details, Pricing, Media, Availability
- Admin configuration with useAsTitle, defaultColumns, listSearchableFields
- Access control: public read, admin create/update/delete
- Status workflow: Draft, Published, Archived
- linkedProperties relationship to Properties collection (hasMany)
- gallery relationship to Media collection (hasMany, sortable)
- availability stored as JSON (Phase 5 will add calendar UI)
- Indexes on status and price

### Collections Index (`payload/src/collections/index.ts`)
- Added Activities to collections array
- Added named export for Activities
